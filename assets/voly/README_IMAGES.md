## Instructions de téléchargement:

1. Allez sur Unsplash.com, Pexels.com ou Pixabay.com
2. Recherchez le nom de la plante en français ou anglais
3. Téléchargez une image de bonne qualité (minimum 500x500px)
4. Renommez l'image selon le nom exact ci-dessus
5. Placez-la dans le dossier `/assets/voly/`
